# TGShevchenko

Мені тринадцятий минало.
Я пас ягнята за селом.
Чи то так сонечко сіяло,
Чи так мені чого було?
І сонце гріло, не пекло!
Та недовго сонце гріло,
Нема в мене хати!
Не дав мені Бог нічого!..
І хлинули сльози,
Тяжкі сльози!.. А дівчина
При самій дорозі
Недалеко коло мене
І поцілувала...
Неначе сонце засіяло,
